# AI To-Do List - React версия

Проект перенесен на React с сохранением структуры SCSS и функционала навигации между вкладками.

## Установка

```bash
npm install
```

## Запуск проекта

```bash
npm run dev
```

Проект будет доступен по адресу `http://localhost:5173`

## Структура проекта

```
src/
├── components/          # Переиспользуемые компоненты
│   ├── Header/         # Компонент шапки
│   └── BottomNav/      # Нижняя навигация
├── pages/              # Страницы приложения
│   ├── Home/           # Главная страница
│   ├── Profile/         # Профиль
│   ├── Settings/        # Настройки
│   ├── Stats/           # Статистика
│   └── Calendar/        # Календарь
└── styles/             # SCSS стили (модульная структура)
    ├── _variables.scss
    ├── _mixins.scss
    ├── _base.scss
    ├── _components.scss
    ├── _layout.scss
    └── style.scss
```

## Функционал

- ✅ Навигация между вкладками (Главная, Статистика, Календарь, Профиль)
- ✅ Переход на страницу настроек из профиля
- ✅ Все остальные функции удалены (как требовалось)

## Технологии

- React 18
- React Router DOM 6
- Vite
- SCSS/Sass
